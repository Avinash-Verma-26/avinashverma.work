# avinash-verma.work

Personal portfolio website for Avinash Verma — design technologist and BIM specialist working at the intersection of architecture, computation, and software.

## About

I build tools that help design teams work faster and smarter: Revit/Rhino automation, workflow add-ins, and data dashboards that turn messy project information into clear insights. This site is a home base for sharing projects, experiments, and ideas that blend design craft with code.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS v4
- React Router

## Pages

- **Home** — intro and blurb
- **Projects** — selected work with case studies
- **CV** — resume viewer with LinkedIn and download links
